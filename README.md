# maybank2u
Extracting 'em transactions

## Setting Up
```
git clone https://github.com/mavx/maybank2u.git
cd selenium

pip install pipenv # Skip this if already installed
pipenv install
```

## Usage
```
pipenv shell
export MAYBANK2U_USERNAME=YOUR_USERNAME
export MAYBANK2U_PASSWORD=YOUR_PASSWORD
python main.py
```
